# Multi-Tabs
A website for bypassing url blockers. Should be used responsibly. 

## How to Use
Different ways to use it. 
**If you want to use it in an about:blank tab, make sure to enable popups from the site you are using (ex: csb.app)**

### Automatic methods
Automatic deployments are now found in the Skibidi-Math fork of Multi-Frames that can be found [here](https://github.com/Skibidi-Math/Multi-Frames)

### Manual method
Go to anywhere where you can interpret html code. Such as [W3 Schools](https://www.w3schools.com/html/tryit.asp?filename=tryhtml_intro) or in files (some districts disable).
Then put everything from index.html into the code area.
